# Squeak Guided Mouse Project
<img width="1163" height="896" alt="Screenshot 2026-03-26 112301" src="https://github.com/user-attachments/assets/3484808a-be2b-4caa-a4fb-3dcb4f78e405" />

This project follows a guided CAD tutorial for designing a custom mouse using the Bambu Mouse Kit files. Unfortunately, a blocky, function-first (fr fr) mouse design was made prioritizing simplicity over aesthetics.

You gotta believe me twin ❤️

## Features
- Mininal design for fast prototyping
- has neccessary components to attach to bambu mouse kit
- hollow, pocketed shell to make the block a bit lighter
- can click 👍

## Future Iterations
I plan to redesign this mouse using form-based modeling (sculpting) to improve ergonomics and create a more natural shape.
